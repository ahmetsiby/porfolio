Présentation

Bonjour ! Je suis Ahmet Siby, un développeur web passionné par la création de sites web modernes, réactifs, performants, sécurisés, et optimisés pour les Progressive Web Apps (PWA). Je maîtrise des technologies telles que HTML, CSS, JavaScript, ainsi que des outils et frameworks back-end comme Node.js. Mon expertise s'étend à la gestion des bases de données, l'optimisation des performances web, la sécurité des applications et la mise en œuvre de PWA pour offrir une expérience utilisateur rapide, fiable et engageante, même hors ligne. Je suis engagé à suivre les bonnes pratiques et à améliorer continuellement les interfaces web.

      Le CSS (Cascading Style Sheets) définit l'apparence des éléments HTML, contrôlant les couleurs, polices, et mises en page. Il sépare la structure du style pour un design flexible

      Bootstrap est un framework CSS open-source qui facilite le développement de sites web réactifs et modernes. Il propose des composants préconstruits et un système de grille flexible pour créer des interfaces rapidement.

    • TailwindCss

      Tailwind CSS est un framework CSS utilitaire qui permet de concevoir rapidement des interfaces en appliquant directement des classes prédéfinies dans le HTML. Il offre une grande flexibilité pour personnaliser les styles sans écrire de CSS personnalisé.
